## 🔍 为什么是Dry Run模式？

### 启动参数

```bash
python scripts/run_lightglow_realtime_trader.py \
  --symbol MNQ \
  --contract-month 202606
  # 没有 --live 参数，默认是 Dry Run
```

### 代码逻辑

```python
if not self.dry_run:
    # 提交订单到IBKR
    order = self.ib.placeOrder(...)
else:
    # 只显示信号，不提交订单
    print("Dry Run: True")
```

## 🔧 如何切换到实盘模式

### 方法1: 使用 --live 参数

```bash
# 停止当前交易器
kill $(cat logs/trader.pid)

# 启动实盘模式
python scripts/run_lightglow_realtime_trader.py \
  --symbol MNQ \
  --contract-month 202606 \
  --live  # 添加这个参数
```

### 方法2: 修改代码默认值

```python
# 在脚本中找到
parser.add_argument("--live", action="store_true", help="Live trading mode")

# 改为
parser.add_argument("--live", action="store_true", default=True, help="Live trading mode")
```
