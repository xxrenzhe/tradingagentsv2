# NQ vs MNQ 合约对比说明

## ⚠️ 重要发现

**回测使用的合约与实时交易器不同！**

---

## 📊 合约对比

### NQ (标准 E-mini Nasdaq 100)

```
合约乘数: $20/点
最小变动: 0.25点 = $5.00
保证金要求: ~$17,000
适合: 大资金账户
```

### MNQ (Micro E-mini Nasdaq 100)

```
合约乘数: $2/点
最小变动: 0.25点 = $0.50
保证金要求: ~$1,700
适合: 小资金账户、测试
```

**关系**: MNQ = NQ / 10

### 如果回测是NQ，实时用MNQ

```

```


## 🔧 如何切换到NQ

如果你想让实时交易器使用NQ（不推荐测试阶段）：

```bash
# 停止当前交易器
kill $(cat logs/trader.pid)

# 启动NQ交易器
source .venv/bin/activate
python -u scripts/run_lightglow_realtime_trader.py \
    --symbol NQ \
    --contract-month 202606 \
    > logs/trader_live_$(date +%Y%m%d_%H%M%S).log 2>&1 &
echo $! > logs/trader.pid
```
