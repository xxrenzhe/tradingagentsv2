## 📊 策略核心理念

### 基本概念

**Premium区域（溢价区）**:
```
定义: 价格分布的上端（95%分位数以上）
市场含义: 价格"过高"，买方力量耗尽
预期: 价格倾向于回落
交易方向: 做空
```

**Discount区域（折价区）**:
```
定义: 价格分布的下端（5%分位数以下）
市场含义: 价格"过低"，卖方力量耗尽
预期: 价格倾向于反弹
交易方向: 做多
```

**中性区域（Fair Value）**:
```
定义: 价格在5%-95%分位数之间
市场含义: 价格处于"公允价值"
预期: 方向不明确
交易方向: 不交易
```

→ 满足所有条件 → 做多1手MNQ
