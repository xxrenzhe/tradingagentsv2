## 🎯 完整交易规则

### 多头入场条件（做多）

**必须同时满足以下5个条件**:

```
1. ATR > 8.0 ✅
   - 含义: 市场波动率足够
   - 原因: 避免在低波动环境交易
   - 参数: 使用优化后的阈值

2. 不在Kill Zone时间 ✅ [新增]
   - 避免时段:
     * NY AM Kill Zone: 8:30 - 11:30 EST
     * NY PM Kill Zone: 13:30 - 16:00 EST
   - 含义: 避开高波动、低质量时段
   - 原因: Kill Zone盈利因子低（1.08 vs 2.45）

3. 价格 < Discount水平 ✅
   - Discount = 过去100根K线的5%分位数
   - 含义: 价格处于"折价"区域
   - 原因: 价格过低，预期反弹

4. 无持仓 ✅
   - 含义: 当前没有持仓
   - 原因: 策略每次只持有1个仓位

5. 其他优化过滤器 ✅
   - 使用walk-forward优化后的参数
   - 可能包括额外的技术指标过滤

→ 满足所有条件 → 做多1手MNQ
```

---

### 空头入场条件（做空）

**必须同时满足以下5个条件**:

```
1. ATR > 8.0 ✅

2. 不在Kill Zone时间 ✅ [新增]
   - 避免: 8:30-11:30 和 13:30-16:00 EST

3. 价格 > Premium水平 ✅
   - Premium = 过去100根K线的95%分位数
   - 含义: 价格处于"溢价"区域
   - 原因: 价格过高，预期回落

4. 无持仓 ✅

5. 其他优化过滤器 ✅

→ 满足所有条件 → 做空1手MNQ
```

---

### 出场条件

**非常简单**:

```
持仓时间 >= 2根K线 → 平仓

示例:
- 10:30入场 → 10:32出场
- 不使用止损

- 固定时间出场

原因:
- 快速进出
- 捕捉短期反转
- 避免持仓过久
```

## 🕐 交易时段详解

### 允许交易的时段（非Kill Zone）

```
时段                    EST时间           特点
================================================================
亚洲时段               18:00 - 02:00     低波动，震荡市，最佳
伦敦时段               02:00 - 08:30     中等波动，良好
盘后时段               16:00 - 18:00     低流动性，可交易
伦敦/纽约重叠          08:30 - 08:30     (仅8:30前)
午休时段               11:30 - 13:30     中等波动，可交易

总计: 约18.5小时/天
```

---

### 禁止交易的时段（Kill Zone）

```
时段                    EST时间           原因
================================================================
NY AM Kill Zone        08:30 - 11:30     盈利因子低（1.08）
NY PM Kill Zone        13:30 - 16:00     盈利因子低（1.08）

总计: 约5.5小时/天

为什么避免Kill Zone？
❌ 盈利因子低: 1.08 vs 2.45（非KZ）
❌ 高波动: 假反转信号多
❌ 强趋势: 反转容易失败
❌ 机构订单: 方向性强，不利反转
❌ 平均每笔低: $8.21 vs $66.49（非KZ）

结论: Kill Zone不适合反转策略
```

## ⚠️ 策略风险

### 主要风险

```
1. 时间止损风险
   - 2根K线可能不够
   - 可能在反转前出场
   - 可能错过大行情

2. 无止损风险
   - 没有价格止损
   - 依赖时间止损
   - 极端行情可能大亏

3. 单一合约风险
   - 只交易MNQ
   - 没有分散
   - 市场特定风险

4. 延迟数据风险（Paper账户）
   - Paper账户数据延迟10-15分钟
   - 实盘可能表现不同
   - 滑点可能更大

5. 时段过滤风险 [新增]
   - 错过Kill Zone的盈利交易
   - 虽然Kill Zone整体盈利因子低
   - 但仍有部分盈利交易
```

---

### 风险管理

```
✅ ATR过滤: 避免低波动环境
✅ Kill Zone过滤: 避开低质量时段 [新增]
✅ 快速出场: 2分钟出场，减少风险暴露
✅ 单一持仓: 避免过度杠杆
✅ 干跑模式: 先验证再实盘
✅ 持续监控: 跟踪实际表现vs回测
```
