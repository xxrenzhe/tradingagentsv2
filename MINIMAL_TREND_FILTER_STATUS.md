## 🎯 过滤规则

### EMA趋势检测

```python
EMA Fast: 20周期
EMA Slow: 50周期

趋势判断:
- fast > slow → 上升趋势
- fast < slow → 下降趋势
- fast ≈ slow → 震荡市场
```

---

### 信号过滤逻辑

```
震荡市场 (EMA中性):
✅ 允许所有信号（保持原始策略）
✅ Premium → 做空
✅ Discount → 做多

上升趋势 (fast > slow):
✅ 允许做多信号（Discount → 做多）
❌ 过滤做空信号（避免逆势）

下降趋势 (fast < slow):
✅ 允许做空信号（Premium → 做空）
❌ 过滤做多信号（避免逆势）
```
