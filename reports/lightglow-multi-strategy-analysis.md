**策略**：1分钟Premium/Discount反向（持仓2分钟）


**优点**：
- ✅ 最高收益

- ✅ 最容易监控

**缺点**：

## 📊 相关性分析

**1. 交易频率太低**

```
Premium/Discount反向（1分钟）：
├── 总交易：38,836笔

├── 总交易：130笔
├── 每年：20笔
└── 每天：0.14笔（一周1笔）

问题：
- 信号太少
- 样本不足
- 无法形成稳定收益
```

**2. 盈利因子不够高**

```
要弥补交易频率差距：

如果Internal OB Break想达到$2M利润：
├── 当前：130笔 = $6.7K
├── 需要：38,836笔才能达到$2M
├── 或者：盈利因子需要达到300+

实际：
├── 盈利因子：1.74
├── 交易频率：130笔
└── 结论：永远无法达到
```

**3. 信号质量问题**

```
Premium/Discount反向：
├── 正向fold率：100%（13/13）
├── 最小净利润：2,125点
└── 稳定性：极高

Internal OB Break：
├── 正向fold率：67%（2/3）
├── 最小净利润：-164点（亏损）
└── 稳定性：差

结论：
- PD反向在所有年份都盈利
- 其他信号在某些年份亏损
- 不可靠
```
