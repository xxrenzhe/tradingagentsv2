## 💰 交易成本配置

### 成本假设

```
1. 手续费 (Commission)
   - 单边: 0.020%
   - 往返: 0.040%
   - 说明: 期货交易的典型手续费

2. 滑点 (Slippage)
   - 单边: 1.0 bps = 0.010%
   - 往返: 2.0 bps = 0.020%
   - 说明: 市价单执行的价格偏差

3. 市场冲击 (Market Impact)
   - 0.5 bps = 0.005%
   - 说明: 订单对市场价格的影响

4. 总成本
   - 每笔交易: 2.54 bps = 0.0254%
   - 计算: 0.040% + 0.020% + 0.005% = 0.0654%
   - 实际: 2.54 bps (基点)
```

### 成本合理性分析

```
我们的假设 (0.0254%/笔):
- 手续费: 0.040% (往返)
- 滑点: 0.020% (往返)
- 市场冲击: 0.005%

行业标准:
- 低成本券商: 0.02-0.05%/笔
- 普通券商: 0.05-0.10%/笔
- 高频交易: 0.01-0.03%/笔

结论: 我们的假设偏保守，实际成本可能更低
```
