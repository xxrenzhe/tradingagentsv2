## 📋 策略逻辑

### 入场条件（全部满足）

```
1. 价格刚刚确认MSS（市场结构突破）
   - 多头: 价格突破swing high（5根K线lookback）
   - 空头: 价格突破swing low

2. 价格在Premium/Discount区域
   - 多头: 在Discount区域（价格低于均衡）
   - 空头: 在Premium区域（价格高于均衡）

3. 趋势方向与区域一致
   - Discount + 向上MSS = 做多 ✅
   - Premium + 向下MSS = 做空 ✅
   - Discount + 向下MSS = 不做 ❌
   - Premium + 向上MSS = 不做 ❌
```

---

### 出场条件（任一满足）

```
1. 反向MSS（趋势结束）
   - 多头: 价格跌破swing low
   - 空头: 价格突破swing high

2. 最大持仓时间（20根K线）
   - 防止无限持仓

3. 合约切换
   - 自动出场
```


## 🔬 实现细节

### MSS检测

```python
def detect_mss_signals(frame, lookback=5):
    for i in range(lookback, len(frame)):
        # 计算swing high/low
        swing_high = high_prices[i-lookback:i].max()
        swing_low = low_prices[i-lookback:i].min()
        
        current_close = close_prices[i]
        
        # 看涨MSS: 突破swing high
        if current_close > swing_high:
            signals[i] = 1
        # 看跌MSS: 突破swing low
        elif current_close < swing_low:
            signals[i] = -1
```

---

### 入场信号组合

```python
# 多头入场: Discount + 看涨MSS
if premium_discount == -1 and mss == 1:
    entry_signal = 1

# 空头入场: Premium + 看跌MSS
elif premium_discount == 1 and mss == -1:
    entry_signal = -1
```

---

### 出场逻辑

```python
for i in range(entry_index, entry_index + max_hold_bars):
    # 检查反向MSS
    if direction > 0 and current_price < swing_low:
        exit()  # 多头遇到看跌MSS
    elif direction < 0 and current_price > swing_high:
        exit()  # 空头遇到看涨MSS
```
