# 组合策略测试 - Kill Zone趋势 + 非Kill Zone反转

**日期**: 2026-05-08  
**测试编号**: #9  
**用户问题**: "那Kill Zone+趋势策略，非Kill Zone+反转策略，能组合成最强的交易策略嘛？"

---

## 🎯 测试假设

### 组合策略设计

```
策略A: Kill Zone时段 → 趋势跟踪
- 时段: NY AM (8:30-11:30) + NY PM (13:30-16:00)
- 方向: 跟随原始信号（不反转）
- 理论: 高流动性，强趋势，适合趋势跟踪

策略B: 非Kill Zone时段 → 反转策略
- 时段: Asian + London + 其他
- 方向: 反转原始信号
- 理论: 低流动性，震荡市，适合反转

组合策略 = A + B
```

---

## 💭 理论基础

### 为什么这可能有效？

```
1. 利用不同时段的特性
   - Kill Zone: 高波动，强趋势
   - 非Kill Zone: 低波动，震荡市

2. 使用适合的策略类型
   - 趋势时段 → 趋势策略
   - 震荡时段 → 反转策略

3. 最大化交易机会
   - 24小时都在交易
   - 每个时段用最优策略

4. 理论上的完美组合
   - Kill Zone趋势跟踪盈利
   - 非Kill Zone反转盈利
   - 两者相加 = 最大利润
```
