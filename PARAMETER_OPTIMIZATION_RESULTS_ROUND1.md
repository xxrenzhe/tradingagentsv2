## 🔍 为什么3分钟表现这么差？

### 原因分析

```
1. 信号频率大幅降低
   - 原始1分钟: ~10,000笔交易
   - 3分钟: 5,811笔交易
   - 减少: 42%
   
2. 错过了最佳入场点
   - 1分钟K线能捕捉精确的Premium/Discount区域
   - 3分钟K线太粗糙，错过最佳入场
   
3. 持仓时间自动延长
   - 原始: 2根K线 = 2分钟
   - 3分钟: 2根K线 = 6分钟
   - 持仓时间延长3倍，增加风险暴露

4. US RTH时段限制
   - 错过了其他时段的盈利机会
   - 原始策略在全天交易表现更好
```

## 🎯 教训

### 1. 更高时间框架不一定更好

```
假设: 3分钟能减少噪音，提高信号质量
现实: 3分钟错过了最佳入场点，降低了盈利能力

教训: 
- 1分钟可能是这个策略的最优时间框架
- Premium/Discount需要精确的入场点
- 更高时间框架会错过机会
```

---

### 2. 时段限制可能有害

```
假设: US RTH时段流动性更好，信号更可靠
现实: 限制时段减少了盈利机会

教训:
- 原始策略在全天交易表现更好
- 可能其他时段也有盈利机会
- 不要过度限制交易时段
```

---

### 3. 持仓时间很关键

```
原始: 2分钟持仓
3分钟: 6分钟持仓（自动延长）

问题:
- 持仓时间延长增加风险暴露

- 6分钟太长，错过最佳出场点
```
